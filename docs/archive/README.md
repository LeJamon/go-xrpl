accurate at the date noted below and are kept for provenance only — they are **not
maintained** and will drift from the current codebase. For current documentation
start at the repository [`README.md`](../../README.md) and the guides in
[`docs/`](../).

afterward; that staleness is exactly why they were archived during the
documentation overhaul (see [`tasks/doc-overhaul.md`](../../tasks/doc-overhaul.md)).
